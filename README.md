# Online Store Sales Dashboard – Excel Project

## Project Overview
This is a student Data Analytics project based on an Online Store dataset. The goal of this project was to clean raw data, analyze business-related questions, and create an interactive Excel dashboard to generate meaningful insights.

## Project Objectives
- Understand and analyze online store sales data
- Perform data cleaning and preprocessing
- Remove duplicate records
- Answer business-related analytical questions
- Build an interactive dashboard using Excel
- Derive insights from sales trends and customer behavior

## Steps Performed

### 1. Data Cleaning
- Checked for missing or inconsistent values
- Standardized data format
- Removed unnecessary data
- Fixed formatting issues

### 2. Duplicate Removal
- Identified duplicate entries
- Removed duplicates to improve data quality

### 3. Data Analysis
Analyzed the dataset based on the provided business questions and identified key trends and insights.

### 4. Dashboard Creation
Created an interactive Excel dashboard using:
- Pivot Tables
- Pivot Charts
- Slicers
- KPIs
- Charts and visualizations

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning Techniques
- Dashboard Design
